Novena semana

Desarrollada del, 04 de Diciembre, al 08 de Diciembre

¿Que temas aprendió?

Se aprendió habilidades fundamentales en React, abarcando la creación de componentes modulares con interfaces definidas, el uso de JSX para estructurar la aplicación mediante código reutilizable, la comprensión del proceso de renderizado en React a través de ReactDOM.render(), la transferencia de datos entre componentes (de padre a hijo, de hijo a padre o entre componentes anidados), y la implementación de una estructura organizada al dividir los componentes en archivos separados para favorecer la reutilización y la mantenibilidad del código. Estas competencias forman una base sólida para el desarrollo efectivo en React.

Definiciones

JSX y React

JSX es una extensión de JavaScript utilizada en React para crear código reutilizable y estructurar la aplicación en base a componentes.

Renderizado en React

React renderiza HTML mediante ReactDOM.render().

Pasar datos entre componentes

Se refiere a la acción de enviar datos entre componentes, ya sea de padre a hijo, de hijo a padre, o entre componentes anidados.

Estructura de componentes en archivos separados

Se recomienda dividir los componentes en archivos separados para facilitar la reutilización del código.

Componente de función y de clase

En React, existen componentes de función y de clase. Los componentes de función devuelven HTML y se escriben con menos código, mientras que los componentes de clase requieren la palabra reservada extends React.Component y un método render().

Procedimientos

  1. Creación de componentes en React: Se crearon componentes en React, los cuales son unidades modulares con interfaces bien definidas que permiten la comunicación entre componentes y la reutilización del código.
  2. Pasar datos entre componentes: Se pasaron datos entre componentes, ya sea de padre a hijo, de hijo a padre, o entre componentes anidados.
  3. Estructura de componentes en archivos separados: Se dividió los componentes en archivos separados para facilitar la reutilización del código.
  4. Uso de JSX en React: Se utilizó JSX, una extensión de JavaScript utilizada en React para crear código reutilizable y estructurar la aplicación en base a componentes.
  5. Renderizado en React: Se utilizó ReactDOM.render() para renderizar HTML en React . Estos procedimientos se llevaron a cabo para aprender y aplicar los conceptos clave de React.

Resultados

  1. Se crearon componentes en React, siguiendo la recomendación de dividir el código en componentes con interfaces bien definidas para la comunicación entre ellos.
  2. Se pasaron datos entre componentes, logrando enviar un objeto con tres propiedades desde el componente 1 al componente 5, donde se visualizaron los datos.
  3. Se dividió los componentes en archivos separados, facilitando la reutilización del código y la organización de la aplicación.
  4. Se utilizó JSX para estructurar la aplicación en base a componentes, lo que permitió crear código reutilizable y renderizar HTML de manera eficiente.
  5. Se logró renderizar HTML en React mediante el uso de ReactDOM.render().
